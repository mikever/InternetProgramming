About This Site

Internet Programming 1 at Santa Fe College

This site is the accumulation of various projects throughout the semester for the completion of the Internet Programming 1 course.

The games on the site are meant to illustrate various javascript programming concepts of simple to medium complexity. Some of the pages and games have the javascript code embeded within the HTML file, while other pages link to external files.

While site design and CSS were not the focus of this course, for the final project the entire site was redesigned to make it a bit more appealing for the user.

Because of the nature of some of the coding for the games, especially where prompts are used, the site works better using the Firefox browser. Other games that do not use prompt inputs should work well on any current browser.
